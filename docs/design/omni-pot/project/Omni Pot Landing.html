四个核心功能

翻译、词典、文字识别、截图翻译——
四个快捷键覆盖全部场景。

每个功能都可绑定一个全局快捷键，也可通过托盘菜单、HTTP API 或剪贴板监听调用。窗口出现在光标处，失焦自动隐藏。

翻译

有选中文本时翻译选中文本；未选中时弹出空输入窗口；开启剪贴板监听时自动翻译新文本。语言自动检测，所有启用引擎并行翻译。

Ctrl+Alt+T
词典

选中字词查询词典释义。英文走英文词典，中文走中文词典，音标、词性、释义、例句结构化展示。

Ctrl+Alt+D
文字识别

区域截图后将图片中的文字提取到识别窗口。支持系统文字识别、Tesseract、百度高精度、腾讯、火山，及 LaTeX 公式与二维码。

Ctrl+Alt+S
截图翻译

一键完成识别 + 翻译。先提取图片中的文字，再直接送入翻译流水线。适合 PDF、漫画、屏幕上的错误信息。

Ctrl+Alt+Shift+S
翻译窗口

多引擎并排，
差异一目了然。

一段文本同时送给多个内置引擎，结果卡片并排展示。哪个译得好，哪个译错了，肉眼可比。

  • 多引擎并行一段文本同时送往多个内置引擎，结果卡片并排呈现
  • 流式输出OpenAI / Gemini / Ollama 等 LLM 服务边写边显示
  • 词典结果结构化音标、词性、释义、例句分段独立展示
  • 结果卡操作集中朗读、复制、折叠集中在每张卡片右上角

跨平台 · 原生体验

在 macOS、Windows 与 Linux 上都像本地工具一样。

基于 Electron 构建，跨平台体验一致。每个平台都做了 frameless 自绘标题栏、系统级文字识别与原生托盘行为的适配。

macOS

frameless 自绘标题栏，Cmd 修饰键，开机自启，调用 Apple Vision 框架文字识别。

11.0+ x64 · arm64 .dmg

Windows

透明无装饰窗口 + 阴影，WinRT 系统级文字识别引擎，托盘单击行为可自定义。

10 / 11 x64 .msi · .exe

Linux

frameless 自绘标题栏，tesseract 调用，Wayland 用户可通过 HTTP API 配合系统快捷键。

X11 · Wayland x64 · arm64 .deb · .rpm · AppImage
