## User

所有字典词典的返回使用相同的格式，DictResult，并且用户可以设置展示多个卡片。服务里面对字典词典分开，改成中文词典，英文词典，可以设置中文使用什么词典，英文使用什么词典。默认情况下中文使用chinese_dictionary和ecdict，英文使用cambridge_dict和 edict。字典词典窗口改成词典。系统托盘里的字典词典也跟着改。
