# 已知问题

## 2026-05-22 反馈

### 1. 置顶按钮竖线在填充颜色时消失
置顶按钮下面的竖线，按下按钮填充颜色时，竖线区域没有被填充，导致竖线视觉上消失。

### 2. 剪贴板监听行为异常
- 即使在系统托盘里没有打开剪贴板监听，应用仍然在监听剪贴板并自动翻译。
- 正确行为：剪贴板监听开关关闭时不应该监听；开启时才自动读取剪贴板内容翻译。
- 应默认开启剪贴板监听。

### 3. 去除空格 / 去除换行符号与 demo 不一致
去除空格和去除换行的符号样式和 demo 设计稿里不一样，需要对齐。

### 4. 文字识别窗口缺少固定按钮
文字识别窗口没有固定按钮，其他窗口有。

### 5. 文字识别右侧标签格式错误
demo 中格式是「识别 简体中文」或「识别 English」（"识别"标签 + 语言），
现在写的是「简体中文的文字识别」，格式完全不对。翻译标签同样的问题：
demo 是「翻译 简体中文」，现在写的是「English 的翻译」。

### 6. 窗口宽度不持久化
用户手动拉伸窗口调整宽度后，下次打开窗口宽度恢复为默认值，没有保存用户设定的宽度。

### 7. 语音朗读声音太小
TTS 朗读音量偏小，需要放大一倍。

### 8. 翻译快捷键说明文本与 demo 不一致
设置中快捷键页面里，翻译快捷键下面的说明文本和 demo 设计稿不一致，需要对照 demo 修正。

### 9. 透明背景开关不即时生效
打开透明背景开关后，已打开的窗口没有立即变为透明，需要重新打开窗口才能生效。应该即时应用到已有窗口。

### 10. 截图翻译结果卡片内容为空
截图翻译后识别成功，但翻译结果只显示一个空卡片，没有内容。

### 11. 截图翻译与文字识别+翻译样式不一致
截图翻译窗口的样式和「先文字识别再点翻译」的样式不一样。两者应该是同一个页面/组件，
文字识别后点翻译只是在右侧多一个翻译结果卡片。截图翻译应该复用文字识别的界面。

### 12. 词典窗口置顶按钮和固定按钮样式与其他窗口不一致
词典页面的置顶按钮和固定按钮的样式跟其他窗口不同。

### 13. 词典搜索结果没有按设置的服务顺序排列
- 搜中文词时出现了 Cambridge 词典（不应该出现，或者顺序不对）。
- 搜英文词时中文词典排在上面。
- 设置里的默认中文词典和默认英文词典已经配置好且顺序正确，但搜索结果没有按照设置的服务和顺序来展示。
