# 前端 Demo 设计稿与当前实现差异分析

> 对比 `docs/design/omni-pot/project/` 下的设计稿原型与 `src/windows/` 下的实际实现。
> 分析日期：2026-05-19

---

## 总览

| 窗口/功能 | 设计稿文件 | 实现文件 | 差异程度 |
|---|---|---|---|
| 翻译窗口 | `windows/translate.jsx` | `src/windows/translate/` | 中等 |
| 文字识别/截图翻译 | `windows/other.jsx` | `src/windows/recognize/index.tsx` | 较大 |
| 词典窗口 | `windows/translate.jsx` (DictWindow) | `src/windows/dict/index.tsx` | 中等 |
| 设置窗口 | `windows/config.jsx` | `src/windows/config/` | 较大 |
| 更新窗口 | `windows/other.jsx` (UpdaterWindow) | `src/windows/updater/index.tsx` | 中等 |
| 托盘菜单 | `windows/other.jsx` (TrayMenu) | `src/windows/tray/index.tsx` | 较大 |
| 截图遮罩 | `windows/other.jsx` (ScreenshotOverlay) | `src/windows/screenshot/index.tsx` | 未详细对比 |
| 共享组件/样式 | `shared.jsx` + `styles.css` | `src/components/` + `src/styles/` | 中等 |

## 一、翻译窗口（TranslateWindow）

### 1.1 Titlebar（标题栏）

| 项目 | 设计稿 | 当前实现 | 差异 |
|---|---|---|---|
| 置顶按钮 (Pin/Thumbtack) | 有，独立控制 | 有 | **一致** |
| 固定按钮 (Lock) | 有，独立控制 | 有 | **一致** |
| 置顶→固定联动 | 置顶开启时自动开启固定 | 有（`alwaysOnTop` 开启时 `pinned = configPinned \|\| alwaysOnTop`） | **一致** |
| Wordmark | "Omni Pot" | "Omni Pot" | **一致** |
| 模式标签 | "翻译" | i18n key `t('translate')` | **一致** |
| 关闭按钮 | 右侧单个 close | 右侧单个 close | **一致** |

### 1.2 Source Area（源文本区域）

| 项目 | 设计稿 | 当前实现 | 差异 |
|---|---|---|---|
| 卡片结构 | `card` 包裹，`padding: 0` | `card` 包裹，`padding: 0` | **一致** |
| 文本区域 | `contentEditable` div | `<textarea>` | 实现方式不同但功能等价 |
| 最大行数 | 8 行自动增长后滚动 | 8 行自动增长后滚动 | **一致** |

| 检测语言显示 | 固定写 "检测为 **英文**" | i18n + 动态语言名 | 实现更完善 |
| 按钮行（底部 action bar） | 去除换行 · 去除空格 · 朗读 · 复制原文 · 清空 · 翻译 | 去除换行 · 去除空格 · 朗读 · 复制原文 · 清空 · 翻译 | **一致** |
| 朗读按钮状态 | idle/busy(loading dots)/playing(brand-filled) 三态 | ttsBusy(loading dots)/ttsPlaying(brand-filled) | **一致** |
| 变量名格式转换 | **无** | 有 `Alt+Shift+U` 快捷键循环转换（snake/camel/pascal 等） | **差异**：实现额外功能，设计稿未涉及 |

| 目标语言列表 | 源语言去掉 auto | 通过配置 | **一致** |

### 1.4 Target Area（翻译结果区域）

| 项目 | 设计稿 | 当前实现 | 差异 |
|---|---|---|---|
| 结果卡片头部 | 拖拽手柄 · 服务图标 · 服务名 · loading dots · 重试 · 朗读 · 复制 · 收藏 · 展开/收起 | 拖拽手柄 · 服务图标 · 服务名 · loading dots · 重试 · 朗读 · 复制 · 收藏 · 展开/收起 | **一致** |

### 2.1 Titlebar

### 2.2 布局（双栏）

| 项目 | 设计稿 | 当前实现 | 差异 |
|---|---|---|---|
| 布局结构 | 1fr 1fr 网格，左边图片右边文字 | 1fr 1fr 网格，左边图片右边文字 | **一致** |

### 2.3 截图翻译模式（translate=true）

| 项目 | 设计稿 | 当前实现 | 差异 |
|---|---|---|---|
| 右侧布局 | 两个纵向卡片（识别结果 + 翻译结果） | **单个 textarea** | **重大差异**：设计稿是两个独立卡片上下排列，实现只有一个可编辑文本框 |
| 识别结果标签 | mono 小字 "识别 · English" | **无** | **缺失** |
| 翻译结果标签 | mono 小字 "翻译 · 简体中文" | **无** | **缺失** |
| 翻译功能 | 识别后自动翻译并显示在下方卡片 | 需点击翻译按钮跳转到翻译窗口 | **重大差异**：设计稿的截图翻译是内联的，实现是跳转到翻译窗口 |

### 2.4 Action Bar（底部操作栏）

| 项目 | 设计稿 | 当前实现 | 差异 |
|---|---|---|---|
| 复制图片按钮 | **有** | **无** | **缺失** |
| OCR 引擎选择 | PillSelect（系统识别/Tesseract/AI视觉/百度高精度） | PillSelect | **一致** |
| 语言选择 | PillSelect（10种语言） | PillSelect | **一致** |
| 截图翻译时 | 显示交换图标 + 目标语言 PillSelect | **无**（跳转翻译窗口） | **缺失** |
| 翻译按钮 | 仅识别模式显示 | 有 | **一致** |
| 去除换行 | 有 | 有 | **一致** |
| 去除空格 | 有 | 有 | **一致** |
| 复制文本 | 有 | 有 | **一致** |
| 导出按钮 | 有（下拉选择格式） | 有 | **一致** |

### 2.5 截图翻译的 Action Bar

设计稿中截图翻译模式的 action bar 与识别模式不同：
```
复制图片 | 系统OCR | 自动检测 | swap | 简体中文 | (spacer) | 去除换行 | 去除空格 | 复制文本 | 导出
```
**实现完全没有这个差异化的 action bar**，截图翻译模式只是跳转到翻译窗口。

## 八、关键缺失功能汇总

按优先级排列：

### P0（重大差异，影响核心 UX）

1. **截图翻译内联翻译**：设计稿中截图翻译窗口右侧是两个卡片（识别结果 + 翻译结果），实现是跳转到翻译窗口。这是核心 UX 差异。
2. **截图翻译 Action Bar**：设计稿中截图翻译模式有独立的 action bar（含目标语言选择），实现完全没有。
3. **Shimmer 加载动画**：翻译结果加载时的骨架屏动画缺失。
4. **托盘缺少"字典词典"入口**：设计稿有独立的字典词典菜单项和快捷键。

### P1（功能差异，影响体验）

5. **设置页面主题选择器**：设计稿用分段控件(ThemeSeg)，实现用下拉。
6. **历史页面搜索和过滤**：设计稿有搜索框、服务过滤、时间过滤，实现缺失。
7. **历史页面 flag 标签**：设计稿用 flag 组件显示语言，实现用纯文本。
8. **备份页面的 WebDAV 测试连接**：设计稿有测试连接按钮，实现缺失。
9. **备份页面最近备份操作**：设计稿有复制/删除按钮，实现缺失。
10. **设置页面服务标签 (tag)**：设计稿有 OFFLINE/PLATFORM 等标签，实现缺失。
11. **设置页面界面语言**：设计稿支持 7 种，实现只支持 2 种。
12. **识别窗口缺少固定(Lock)按钮**。
13. **词典窗口单词头部缺少朗读按钮**。
14. **词典窗口缺少 CEFR 等级标签**。

### P2（样式/细节差异）

15. **更新窗口 noPin 设计**：设计稿 noPin，实现有 pin 按钮。
16. **更新窗口内置下载 vs 跳转浏览器**。
17. **托盘菜单分隔线分组**。
18. **托盘菜单行样式**（圆角、padding）。
19. **设置备份目标选择**：设计稿用卡片按钮，实现用下拉。
20. **词典结果卡片不支持折叠**。
21. **设置侧边栏版本号**。
22. **识别窗口图片卡片背景色**：设计稿用暖色 `#f7f5f0`，实现用 CSS 变量。
23. **翻译窗口字体大小可配置 vs 设计稿固定 13.5px**。
24. **设置 API 端口缺少 "?" 帮助链接**。
