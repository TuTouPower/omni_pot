# Omni Pot User Guide

> [中文版](../cn/README.md)

Omni Pot is a cross-platform desktop translation, OCR, and dictionary tool. It lives in the system tray and can be invoked at any time via global hotkeys.

## Features

| Feature | Description |
|---|---|
| [Translation](translation.md) | Selection translation, input translation, clipboard monitoring. Multiple engines run in parallel with results shown as side-by-side cards. |
| [Dictionary](dictionary.md) | Look up selected words in English or Chinese dictionaries with pronunciation, part of speech, and definitions. |
| [OCR](ocr.md) | Screenshot and recognize text with Tesseract, system OCR, or cloud OCR services. |
| [Screenshot Translation](ocr.md#screenshot-translation) | Screenshot, recognize text, and auto-translate in one flow. |

## Quick Start

1. [Download and install](installation.md) the package for your platform
2. On first launch, a welcome page shows hotkey guidance
3. Set up hotkeys in [Hotkey Settings](settings.md#hotkeys) for translation, dictionary, OCR, and screenshot translation
4. Select any text and press the hotkey to use
