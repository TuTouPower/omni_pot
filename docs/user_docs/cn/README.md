# Omni Pot 使用手册

> [English version](../en/README.md)

Omni Pot 是一款跨平台桌面翻译、文字识别和词典工具。常驻系统托盘，通过快捷键随时调用。

## 功能概览

| 功能 | 说明 |
|---|---|
| [翻译](translation.md) | 划词翻译、输入翻译、剪贴板监听翻译，支持多个翻译引擎并行对比 |
| [词典](dictionary.md) | 选中单词查询英文词典或中文词典，显示音标、词性、释义 |
| [文字识别](ocr.md) | 区域截图识别文字，支持 Tesseract、系统 OCR、云端 OCR |
| [截图翻译](ocr.md#截图翻译) | 截图识别后自动翻译，识别与翻译联动 |

## 快速开始

1. [下载安装](installation.md)对应平台的安装包
2. 首次启动会显示欢迎页，展示快捷键引导
3. 在[快捷键设置](settings.md#快捷键)中绑定翻译、词典、文字识别、截图翻译的快捷键
4. 选中任意文字，按下快捷键即可使用
