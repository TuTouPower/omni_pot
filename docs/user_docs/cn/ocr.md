# 文字识别与截图翻译

Omni Pot 支持截图后识别文字（文字识别）和截图后自动翻译（截图翻译）。

## 文字识别

1. 按下文字识别快捷键（需先在设置中绑定）
2. 屏幕出现截图覆盖层，拖拽选择要识别的区域
3. 按 `Enter` 或松开鼠标确认选区；按 `Escape` 或右键取消
4. 文字识别窗口弹出，左侧显示截图原图，右侧显示识别结果

### 截图操作

- **拖拽**：创建选区，显示主色描边和尺寸标签
- **Enter / 松开鼠标**：确认选区
- **Escape / 右键**：取消截图

### 识别窗口布局

窗口分三排：

1. **顶部栏**：模式标签（文字识别）、置顶/固定按钮、关闭按钮
2. **中间**：左侧截图原图（大卡片），右侧两个小卡片纵向排列 — 识别内容卡片（可编辑）和翻译内容卡片
3. **操作栏**：
   - 左侧：复制图片 | 选择识别引擎 | 自动检测
   - 右侧：翻译 | 去除换行 | 去除空格 | 复制识别文本 | 导出

### 支持的识别引擎

| 引擎 | 说明 |
|---|---|
| Tesseract | 本地 OCR，支持多语言（按语言包下载） |
| 系统 OCR | Windows WinRT / macOS Vision，支持中英文等 |
| 二维码识别 | 本地识别二维码内容 |
| 百度 OCR | 需要 client_id + client_secret |
| 腾讯 OCR | 需要 secretId + secretKey |
| 火山 OCR | 需要 appid + secret |
| 讯飞 OCR | 需要 appid + apisecret + apikey |
| AI 视觉 | OpenAI 兼容视觉模型，需要 API Key |

## 截图翻译

截图翻译 = 截图识别 + 自动翻译，是文字识别的延伸模式。

1. 按下截图翻译快捷键（需先在设置中绑定）
2. 拖拽选择要识别的区域
3. 识别结果自动发送到右侧的翻译内容卡片，同时翻译

### 按钮差异

截图翻译模式的语言栏增加翻译目标语言选择（"自动检测 → 简体中文"），底栏不单独显示"翻译"按钮。

### 自动联动

- 切换识别语言：自动重新识别并刷新翻译
- 切换翻译目标语言：自动重新翻译（不重新识别）
- 目标语言一旦确立，识别语言变更不改变目标语言

## 导出

支持将识别结果导出为以下格式：

- Markdown (.md)
- 纯文本 (.txt)
- Word 文档 (.docx)
- Word 97 (.doc)

## 文字识别设置

在 **设置 → 文字识别** 中配置：

| 设置 | 说明 |
|---|---|
| 默认识别引擎 | 选择默认使用的 OCR 服务 |
| 默认识别语言 | 默认为自动检测 |
| 自动去除换行 | 识别后自动规范化换行（默认关闭） |
| 自动复制结果 | 识别完成后自动复制文本（默认开启） |
