# 词典

Omni Pot 的词典功能按输入语言自动路由到对应词典：英文查英文词典，中文查中文字典。

## 使用方式

1. 选中文字，按下词典快捷键（需先在设置中绑定）
2. 或在系统托盘菜单中点击"词典"
3. 词典窗口弹出，显示查询结果

也可在词典窗口内直接输入文字查询。

## 词典窗口

### 源词卡片

顶部显示查询词，可直接编辑文字，按 `Enter` 重新查询。

- 左侧：语言检测标签（如"检测为 English"）
- 右侧：复制单词、查询按钮

### 结果卡片

根据输入语言调用不同词典：

**英文输入**：
- 剑桥词典（Cambridge Dictionary）— 提供音标、词性、释义、例句

**中文输入**：
- Chinese Dictionary — 本地离线词典，320K 词语、16K 单字、50K 成语
- CC-CEDICT — 离线英汉词典

每张卡片内容：
- **音标/发音**：显示在卡片内（英文词典有原声时可播放）
- **释义**：按词性分组，每组带编号义项
- **例句**：左侧细线分隔，源文/译文双行

中文词典卡片不显示词性标签和朗读按钮。

## 词典设置

在 **设置 → 服务** 中管理词典服务：

- **中文词典** Tab：管理中文词典服务实例，可启停、排序
- **英文词典** Tab：管理英文词典服务实例，可启停、排序

Chinese Dictionary 为本地离线词典（86MB SQLite 数据库），无需网络和密钥。如需节省磁盘空间，可在设置中关闭。
