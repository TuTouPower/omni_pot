# 常见问题

## 安装与启动

### Q: macOS 打开时提示"无法验证开发者"

macOS 对未公证的应用有此限制。前往 **系统设置 → 隐私与安全性**，在"安全性"区域点击"仍要打开"。

### Q: Linux AppImage 无法运行

确保文件有执行权限：

```bash
chmod +x OmniPot-{version}-linux.AppImage
```

### Q: 便携版和安装版有什么区别？

| 对比项 | 安装版 | 便携版 |
|---|---|---|
| 安装 | 需要安装，写入注册表 | 双击即用 |
| 更新 | 自动更新替换 | 自动更新替换 |
| 配置位置 | `%APPDATA%\omni_pot\` | `%APPDATA%\omni_pot\` |
| 卸载 | 控制面板卸载 | 删除 exe 文件 |

两者的配置数据存放位置相同。

### Q: Wayland 下快捷键不生效

Wayland 环境下 Electron 全局快捷键可能不可靠。替代方案：

- 使用 HTTP API（默认端口 20202）
- 通过系统托盘菜单操作

## 翻译

### Q: 翻译结果为空或报错

1. 检查网络连接
2. 在 **设置 → 服务 → 翻译** 中测试对应服务是否正常
3. 部分服务需要 API 密钥，确认密钥已正确填写

### Q: 翻译窗口消失太快

默认未固定时失去焦点会自动关闭。解决方案：

- 点击窗口左上角"固定"按钮
- 或在 **设置 → 翻译 → 窗口** 中关闭"失焦时关闭"

### Q: 检测语言不准确

语言检测使用本地引擎，对短文本可能不准确。手动选择源语言可绕过此问题。

### Q: 如何切换翻译引擎的顺序？

在 **设置 → 服务 → 翻译** 中拖拽服务实例调整顺序，排在前面的优先显示结果。

## 词典

### Q: 中文词典显示"加载中"

Chinese Dictionary 是本地离线词典（86MB），首次使用需要下载数据库文件。确保应用目录下有 `chinese_dictionary.db` 文件。

### Q: 词典窗口不弹出

确认已绑定词典快捷键（**设置 → 快捷键 → 词典**），并确认选中了文字。

---

## 文字识别

### Q: Tesseract 识别结果不准确

Tesseract 的识别准确度取决于图片质量和语言。建议：

- 确保截图清晰，文字足够大
- 在识别引擎下拉中选择正确的语言
- 对于中文，使用系统 OCR（Windows/macOS）通常比 Tesseract 更准确

### Q: 系统 OCR 不可用

系统 OCR 仅支持 Windows 和 macOS。Linux 用户请使用 Tesseract。

- Windows：需要 Windows 10+，调用系统内置的 WinRT OCR 引擎
- macOS：调用系统 Vision 框架

### Q: 如何切换截图翻译的目标语言？

在截图翻译窗口的底部操作栏，点击语言区域（"自动检测 → 简体中文"）可切换语言。切换后自动重新翻译。

## 设置与配置

### Q: 配置文件在哪里？

| 平台 | 路径 |
|---|---|
| Windows | `%APPDATA%\omni_pot\config.json` |
| macOS | `~/Library/Application Support/omni_pot/config.json` |
| Linux | `~/.config/omni_pot/config.json` |

也可在 **设置 → 关于 → 设置目录** 处查看并复制路径。

### Q: 日志文件在哪里？

与配置目录同级的 `logs/main.log`。也可在 **设置 → 关于** 点击"导出日志"打包最近 7 天的日志。

### Q: 如何备份配置？

在 **设置 → 备份** 中点击"立即备份"，选择保存位置。备份包含配置和翻译历史记录，但不包含 API 密钥等凭据。

### Q: 更新后配置丢失？

Omni Pot 的更新不覆盖用户配置。如遇配置丢失，可在 **设置 → 备份** 中从备份恢复。

---

## 系统托盘

### Q: 如何退出 Omni Pot？

Omni Pot 常驻系统托盘，关闭所有窗口不会退出。右键点击托盘图标，选择"退出"。

### Q: 托盘图标不显示

- Windows：检查任务栏右侧的溢出区域，可能被折叠
- macOS：检查菜单栏右侧
- Linux：确认系统支持系统托盘（部分桌面环境如 GNOME 需要扩展）

### Q: 剪贴板监听消耗资源吗？

剪贴板监听通过轮询实现，轮询间隔较长，对系统资源影响极小。不需要时可在托盘菜单或设置中关闭。
