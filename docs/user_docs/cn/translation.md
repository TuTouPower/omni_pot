# 翻译

Omni Pot 的翻译功能支持多种使用方式：划词翻译、输入翻译、剪贴板监听翻译。一次翻译同时调用多个引擎，结果以卡片并列展示。

## 使用方式

### 划词翻译

1. 在任意应用中选中文字
2. 按下翻译快捷键（需先在设置中绑定）
3. Omni Pot 弹出翻译窗口，显示所选文字的翻译结果

### 输入翻译

未选中文字时按下翻译快捷键，或通过系统托盘菜单点击"翻译"，打开空的翻译窗口，直接输入文字并按 `Enter` 翻译。

### 剪贴板监听

在设置或系统托盘中开启"剪贴板监听"后，每次复制新文本到剪贴板，Omni Pot 自动弹出翻译窗口并翻译该文本。

> 剪贴板监听关闭时完全不监听，不会消耗系统资源。

## 翻译窗口

翻译窗口由三个区域组成：

### 源文本区域

- 文本输入框自动增长，最多 8 行，超过后输入框内部滚动
- 检测到的语言标签（如"检测为 English"）以主色显示
- 操作按钮：**去除换行** | **去除空格** | **朗读** | **复制原文** | **清空** | **翻译**
- `Enter` 触发翻译，`Shift+Enter` 换行，`Escape` 关闭窗口

### 语言区域

居中显示：自动检测 → 简体中文

- 点击语言名称可切换源语言或目标语言
- 点击中间的转换符号可交换源语言和目标语言

### 结果区域

- 每个已启用的翻译服务显示一张结果卡片
- 等待结果时卡片折叠，显示轻量翻译中动效
- 结果返回后卡片自动展开，窗口高度随之增长
- 卡片右上角操作：**朗读** | **复制** | **折叠**
- 翻译失败时显示红色错误信息和重试按钮
- 支持拖拽排序，排序自动保存

## 窗口行为

| 设置 | 说明 |
|---|---|
| 固定 | 开启后失去焦点不会自动关闭 |
| 置顶 | 窗口始终在最前面（自动开启固定） |
| 记住窗口大小 | 记住用户调整的宽度，下次打开时恢复 |
| 窗口位置 | 鼠标位置 / 上次位置 |

## 翻译设置

在 **设置 → 翻译** 中可配置：

| 设置 | 说明 |
|---|---|
| 源语言 / 目标语言 | 默认翻译方向，支持自动检测 |
| 第二语言 | 检测到的语言与目标语言相同时回退使用 |
| 自动复制 | 翻译成功后自动复制译文 |
| 增量翻译 | 新文本追加到现有源文本末尾 |
| 动态翻译 | 输入时自动翻译（1 秒防抖） |
| 自动去除换行 | 传入文本自动规范化换行和连字符断行 |
| 禁用历史记录 | 不保存翻译记录 |

## 支持的翻译服务

Omni Pot 内置 19 个翻译服务引擎，可在 **设置 → 服务 → 翻译** 中管理：

| 服务 | 说明 |
|---|---|
| 必应翻译 | 免费，无需密钥 |
| 谷歌翻译 | 免费，无需密钥 |
| DeepL | 支持 Free / API / DeepLX |
| MyMemory | 免费，无需密钥 |
| 百度翻译 | 需要 AppID + 密钥 |
| 有道翻译 | 需要 AppID + 密钥 |
| 阿里翻译 | 需要 AccessKey |
| 腾讯翻译 | 需要 SecretId + SecretKey |
| 火山翻译 | 需要 AppID + 密钥 |
| 彩云小译 | 需要 Token |
| 牛翻译 | 需要 API Key |
| ChatGLM | 需要 API Key，支持流式输出 |
| Gemini Pro | 需要 API Key，支持流式输出 |
| Ollama | 本地大模型，无需密钥 |

默认启用：必应翻译、DeepL、MyMemory。
