# 设置详解

Omni Pot 的设置窗口可通过系统托盘菜单打开，或点击设置快捷键。

设置采用左侧导航栏 + 右侧内容区的布局。

## 通用

### 应用

| 设置 | 说明 |
|---|---|
| 开机自启 | 登录后自动启动 Omni Pot |
| 启动时检查更新 | 开启后每次启动自动检查新版本 |
| 本地 API 端口 | HTTP API 服务端口（默认 20202），修改后需重启应用 |
| 界面语言 | Omni Pot 界面语言（中文、English 等） |

### 外观

| 设置 | 说明 |
|---|---|
| 主题 | 跟随系统 / 浅色 / 深色（三选一按钮） |
| 主色调 | 陶土橙、群青、松绿、芥末、天蓝（默认）五种主题色 |
| 透明背景 | 窗口背景透明 |
| 托盘左键点击行为 | 打开设置 / 打开翻译 / 无操作 |

## 翻译

### 语言

| 设置 | 说明 |
|---|---|
| 源语言 | 默认源语言，可设为自动检测 |
| 目标语言 | 默认翻译目标语言 |
| 第二语言 | 检测到的语言与目标语言相同时回退到此语言 |

### 行为

| 设置 | 说明 |
|---|---|
| 自动复制 | 翻译完成后自动复制译文到剪贴板 |
| 增量翻译 | 新文本追加到现有源文本末尾（默认替换） |
| 动态翻译 | 输入时自动翻译，1 秒防抖 |
| 自动去除换行 | 传入文本自动规范化换行和连字符断行 |
| 禁用历史记录 | 翻译后不保存到历史记录 |

### 窗口

| 设置 | 说明 |
|---|---|
| 窗口位置 | 鼠标位置 / 上次位置 |
| 记住窗口大小 | 记住用户调整的宽度 |
| 失焦时关闭 | 未固定时失去焦点自动关闭 |
| 始终置顶 | 翻译窗口始终在最前面 |
| 隐藏源文本 | 划词翻译时隐藏源文本区域 |
| 隐藏语言选择 | 隐藏语言转换区域 |
| 翻译后隐藏窗口 | 翻译完成后自动隐藏窗口 |

## 文字识别

| 设置 | 说明 |
|---|---|
| 默认识别引擎 | 选择默认 OCR 服务 |
| 默认识别语言 | 默认为自动检测 |
| 自动去除换行 | 识别后自动规范化换行（默认关闭） |
| 自动复制结果 | 识别完成后自动复制文本（默认开启） |

## 快捷键

四个全局快捷键绑定：

| 快捷键 | 说明 |
|---|---|
| 翻译 | 选中文本时翻译；未选中时弹出空输入窗口 |
| 词典 | 选中文本时查询词典 |
| 文字识别 | 启动截图识别 |
| 截图翻译 | 启动截图翻译 |

每个快捷键有绑定/解绑按钮。绑定时捕获组合键（按 Backspace 清除）。快捷键被其他应用占用时，在状态区域以红色提示。

快捷键展示按平台显示修饰键：Windows/Linux 显示 `Ctrl`，macOS 显示 `Cmd`。

## 服务

五个 Tab 切换管理五类服务：

| Tab | 说明 |
|---|---|
| 翻译 | 翻译引擎实例列表，支持启停、排序 |
| 中文词典 | 中文词典服务实例 |
| 英文词典 | 英文词典服务实例 |
| 文字识别 | OCR 服务实例 |
| 语音朗读 | TTS 服务实例 |

每个实例支持：拖拽排序、启用/停用、编辑配置、删除。同一服务可创建多个实例，配置各不相同。

## 历史

- 翻译历史记录表格：服务图标、源文本、源/目标语言、译文、时间戳
- 工具栏一行展示：启用开关、搜索框、服务筛选、时间筛选、清空按钮
- 支持分页浏览、点击行查看详情并编辑
- 启用开关关闭时，新翻译不写入历史，历史操作置灰

## 备份

- **备份**：将配置（不含密钥）和历史记录打包为本地 zip 文件
- **恢复**：从 zip 备份中恢复配置和历史记录
- 备份 zip 可重新导入，恢复后配置和历史记录正确
- API 密钥等凭据不包含在备份中，恢复后需用户重新填写

## 关于

显示 Omni Pot 版本、官网/文档/反馈/支持作者链接、诊断信息：

| 项目 | 说明 |
|---|---|
| 日志目录 | 主进程运行日志位置，附带复制路径按钮 |
| 设置目录 | 用户数据目录（config.json 所在位置），附带复制路径按钮 |
| 本机 API | HTTP API 地址，附带复制按钮 |
| API Token | API 认证令牌，附带复制按钮 |
| 导出日志 | 最近 7 天的日志打包为 zip，可附在反馈中 |
