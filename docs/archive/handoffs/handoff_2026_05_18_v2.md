## 七、可能踩的坑

- **`native_language_name()` vs `language_name()`**：前者返回语言的本地名称（如"日本語"），后者返回 i18n 翻译（中文环境下也是中文）。所有面向用户的语言名称显示都应用 `native_language_name()`。
- **服务统计 bug 的根因**：`{serviceList.length}` 在 tab 渲染中用了当前 tab 的 reactive state，所有 tab 显示同一个数。修复方式是预计算每个 category 的 count。
- **翻译卡片折叠逻辑**：`manuallyCollapsedKeys` 和自动展开的 `useEffect` 需要配合 `isTranslating` 状态。新翻译开始时清空 `manuallyCollapsedKeys`，结果到达时自动从 `manuallyCollapsedKeys` 中删除对应 key。
- **`npm run dist` 可能被 Omni Pot 进程锁住**：如果已有 Omni Pot 在运行，打包可能失败。需要先关闭或等打包脚本自动处理。
- **词典数据库路径**：`electron/chinese_dict/index.ts` 中 `find_db_path()` 检查多个候选路径。中文词典搜不到可能是数据库文件缺失，需排查。
